# 🎉 Руководство по установке

Добро пожаловать! На этой странице демонстрируется процесс установки [системы управления ресторанами Nutrix](/ru/). Давайте взаимодействуем с вашим рестораном! 🚀

::: info 💡 Внимание!
Обратите внимание, что эти пакеты установки предлагают **минимальную установку**, необходимую для запуска системы, которая *не включает* функции продвинутой аутентификации прямо из коробки.

Хотите запереться с аутентификацией? 🔐 Пожалуйста, ознакомьтесь со страницей документации [Аутентификация](/ru/docs/authentication) для полной информации!
:::

## 🛠️ Предварительные условия

NutrixPOS любит **MongoDB** 🍃 для хранения всех ваших вкусных заказов, инвентаря, продаж и других важных данных! 

Вам потребуется запущенный экземпляр сервера MongoDB перед началом. 
👉 Получите его со страницы официальной загрузки: [http://mongodb.com/try/download/community](http://mongodb.com/try/download/community)

## 📥 Загрузка

Пора достать товар! Посетите нашу [страницу загрузки](https://github.com/nutrixpos/pos/releases/latest), чтобы найти последнюю и величайшую версию NutrixPOS. ✨

💻 **Для пользователей Windows:** ищите файл, заканчивающийся **`windows64.exe`**, и нажмите загрузку!

## 🪄 Процесс установки

У вас есть исполняемый файл? Потрясающе! Убедитесь, что ваш MongoDB включен и работает, затем просто дважды щелкните по файлу установки NutrixPOS и следуйте простым подсказкам. 

После завершения установки запустите свежеустановленное приложение NutrixPOS! 

🪟 **Совет для Windows:** Вы можете легко найти его в меню "Пуск" под **Nutrixpos** или использовать новый блестящий ярлык на рабочем столе.

### ⚙️ Первоначальная установка

Когда вы открываете NutrixPOS в первый раз, вам будут представлены этот экран настройки:

![Установка NutrixPOS](https://elmawardy.sirv.com/nutrixpos-docs/nutrixdocs3.png)

1. **Подключите точки:** Добавьте информацию о подключении к базе данных в обозначенные поля.
2. **Протестируйте:** Нажмите кнопку **Проверить соединение** 🧪.
3. **Сохраните:** Если тест пройдет успешно, кнопка **Сохранить и продолжить** загорится! 🌟 Щелчок по нему сохранит конфигурацию вашей базы данных в безопасности в файле `config.yaml`.
4. **Поехали!** Появится новая кнопка **"Поехали"**. Дайте ему щелчок, и вы перейдете прямо на страницу кассира, которая выглядит примерно так:

![Установка NutrixPOS](https://elmawardy.sirv.com/nutrixpos-docs/nutrixdocs4.png)
