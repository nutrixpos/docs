# Installation

This page demonstrates the installation process of the [nutrix restaurant management system](/).


## Windows

For windows installation visist the [Download](https://github.com/nutrixpos/pos/releases/latest) page and download the file with the postfix **windows64.exe**

::: info
Please note that the current installation packages offer the minimal installation required to run the system, which is the nutrixpos app + the mongodb which don't include the authentication features.

If you want to add authentication kindly check the [Authentication](/docs/authentication) docs page
:::
